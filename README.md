# How to start to make this page
[doc](http://jekyllrb-ja.github.io/docs/)

## setup
```
$ gem install jekyll bundler
$ clone git@github.com:abc-research/challenge2022.git
$ cd challenge2022
$ bundle install
```

## run surver
```
$ bundle exec jekyll serve
```
